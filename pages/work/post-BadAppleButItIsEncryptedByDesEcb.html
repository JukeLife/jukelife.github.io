TIP

　Bashが利用できる場合、xargsコマンドを使用すると複数のスレッドを用いて効率的に処理できます。

Bash
$ find "<入力フォルダー>" -name "*.<拡張子>" | sed 's!^.*/!!' | xargs -P <スレッド数> -i java -jar DES_Visualizer.jar "<入力フォルダー>/{}" "<出力フォルダー>/{}" -1 -1 128 "ECB" "BADAPPLE"
